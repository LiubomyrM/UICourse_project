Knowledge evaluation portal with the following features.
-building custom tests through an online form
-custom tests import via CSFs or JSON or anything else (for e.g. xml)
-detailed result reports with graphs
-final result interactive options for sharing, saving as PDF, emailing...
-try/catch/throw based hinting system (optional)
-supplementary literature/course(including "LITS :)) suggestions (dynamically built depending on which areas where the most difficult for the user)
-ability to re-use/edit/improve/copy/rate existing tests
